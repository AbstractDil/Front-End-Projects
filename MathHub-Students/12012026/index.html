Algebraic Identities - Master Sheet

I. Square Identities

Fundamental Forms Expansion:

1. (a + b)2 = a2 + 2ab + b2

2. (a - b)2 = a2 - 2ab + b2

3. (a2 - b2) = (a + b)(a - b) ← Diff of Squares!

Useful Combinations & Rearrangements:

i. 2(a2 + b2) = (a+b)2 + (a-b)2

ii. 4ab = (a + b)2 - (a - b)2

iii. ab = (a+b/2)2 - (a-b/2)2

iv. a2 + b2 = (a + b)2 - 2ab

v. a2 + b2 = (a - b)2 + 2ab

Switching between Sum and Difference:

4. (a + b)2 = (a - b)2 + 4ab

5. (a - b)2 = (a + b)2 - 4ab

Triple Variable Square:

(a + b + c)2 = a2 + b2 + c2 + 2ab + 2bc + 2ca

II. Cube Identities

6. (a + b)3 = a3 + b3 + 3ab(a + b)

7. (a - b)3 = a3 - b3 - 3ab(a - b)

Common Factorized Forms:

8. (a + b)3 = a3 + b3 + 3ab(a + b)

9. (a - b)3 = a3 - b3 - 3ab(a - b)

10. a3 + b3 = (a + b)3 - 3ab(a + b) ← Alternative Sum!

Factorization:
a3 + b3 = (a + b)(a2 - ab + b2)
a3 - b3 = (a - b)(a2 + ab + b2)
Check signs!
(a+b) has -ab
(a-b) has +ab

Triple Variable Cube:

(a + b + c)3 = a³ + b³ + c³ + 3(a+b)(b+c)(c+a)

*Note: This can also be expanded to terms like 3a²b, 3a²c etc., but this factored form is easier to recall!

III. Conditional Identity

a³ + b³ + c³ - 3abc = (a+b+c)(a² + b² + c² - ab - bc - ca)

IMPORTANT: If a + b + c = 0, then...
a³ + b³ + c³ = 3abc
